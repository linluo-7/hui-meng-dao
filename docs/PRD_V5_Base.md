# 绘梦岛 App — 需求规格基线（由 PDR V5.0 固化）

**文档角色**：产品/研发/测试/运营的唯一基线（后续迭代与验收以本文件为准）  
**来源**：用户提供《绘梦岛 App 产品需求文档（PDR）V5.0（2026-03-19）》  
**适用阶段**：开发前 → 全周期  
**范围**：纯软件功能需求、交互、规则、数据约束、技术要求、验收口径  

> 约定：本文件不讨论 UI 视觉稿，不讨论商业/团队管理流程。  
> 术语：OC=原创角色；企划组=企划发起/管理者；玩家=参与者；内场/外场=企划权限分区；Bot=NPC/GM 自动化系统。

---

## 1 产品概述

### 1.1 定位
为绘圈/OC 创作者提供**一站式企划管理 + 社交互动**的垂直社区平台。

### 1.2 愿景
让企划组高效开设与运营企划；让玩家沉浸式参与、作品可沉淀、有归属感；成为绘圈企划首选平台。

### 1.3 用户与痛点（用于设计约束）
- **核心用户**：OC 创作者、企划爱好者、跑团玩家（15–28，偏高中/大学）
- **企划组痛点**：管理繁琐/信息分散/玩家质量难控/企划易烂尾/规则难验证/成员负荷不可视
- **参与者痛点**：组织混乱/规则不清/审核不透明/作品难沉淀/社交纠纷/信用缺失/角色关系难追踪

## 2 功能需求（全量覆盖，禁止遗漏）

### 2.1 用户系统

#### 2.1.1 注册与登录
- **唯一登录方式**：手机号 + 验证码
- **必须支持**：登录态保持、退出登录、账号注销
- **强制约束**：真实用户校验（不允许第三方登录）

#### 2.1.2 个人主页
- **展示**：头像、昵称、简介、作品、参与企划、关注/粉丝列表
- **权限**：公开/私密权限设置
- **数据看板**：作品数、企划数、信誉分、获得称号

#### 2.1.3 社交关系
- **关注体系**：关注、粉丝、互关展示
- **黑名单**：屏蔽发言、访问、互动、私信
- **私信权限**：所有人 / 关注 / 好友可控

---

### 2.2 角色管理（OC 核心）

#### 2.2.1 我的角色
- **多角色**：支持创建多个 OC
- **字段**：头像、名称、设定、形象图、背景故事
- **操作**：新增、编辑、删除、隐藏

#### 2.2.2 马甲功能（企划内身份）
- **绑定规则**：每个企划可绑定一个独立 OC
- **展示规则**：企划内发言以角色身份展示，与主账号区分

#### 2.2.3 角色档案
- **记录**：参与企划、剧情事件、关联作品
- **权限**：公开 / 仅好友 / 仅自己可见

#### 2.2.4 角色公开与关注
- **公开开关**：可设置角色为公开状态
- **关注角色**：其他用户可关注角色并接收角色动态推送

#### 2.2.5 动态关系图谱
- **可视化**：展示角色人际关系演变
- **关系类型**：盟友/对立/暧昧/亲属等（可扩展）
- **变更记录**：标注剧情节点、时间、来源
- **时间线**：支持回溯查看

#### 2.2.6 角色排行榜
- **排序依据**：参与企划次数
- **展示限制**：仅前 N 名（N 后台可配置）

---

### 2.3 企划管理（核心模块）

#### 2.3.1 企划创建与本体构建
1) **标准化企划主页模板**
- 预设模块：背景设定、主旨、主线/支线、玩法、人设要求、规则
- 自动排版：字体、行距、配色、布局优化
- 支持自定义增删模块

2) **世界观构建工具**
- 世界观文档
- 地图上传与标注
- 人物关系图谱、势力分布编辑
- 可关联到企划主页

3) **NPC 档案管理**
- NPC 角色卡：定位、作用、性格、关系、形象
- 关联剧情节点，供 Bot 调用

4) **主线任务系统**
- 按章节发布主线内容
- 支持分支剧情、阶段性总结
- 记录玩家行为对剧情的影响

5) **支线活动发布器**
- 类型：节日、配对、投票、接龙、问答、集体 ASK
- 配置：参与门槛、人数、时间限制
- 一键发布，自动统计结果

6) **自定义游戏机制编辑器**
- 支持配置：PVP、掷骰、判定、数值、概率、冷却
- 内置模板：d100、rd6、大逃杀 PVP 等
- **规则复用**：支持（需授权）
- 原创声明与版权标记

7) **沙盒测试模式**
- 虚拟角色、虚拟场景
- 完整模拟 PVP/PVE 流程
- 检测：逻辑漏洞、数值失衡、流程卡死
- **隔离要求**：测试数据不写入正式环境

#### 2.3.2 玩家招募与审核
1) **可配置人设卡模板**
- 字段类型：文本、图片、文件
- 约束：必填/选填、格式约束
- 可上传示例图
- **文件限制**：大小 < 5M（强制）

2) **审核系统**
- 多人协作审核
- 支持：评分、投票、文字反馈
- 可配置客观评分标准
- 投递渠道：站内私信/邮箱投递
- 批量操作：批量通过、批量打回、批量分配任务

3) **报名人数控制与分流**
- 最大报名数，满员自动关闭
- 内外场分流：外场仅可查看公开内容
- **转内场申请通道**：必须提供

4) **智能 FAQ 与答疑**
- FAQ 库：分类、搜索
- 玩家提问自动匹配 FAQ
- 匿名提问箱
- 记录高频问题并推送企划组

#### 2.3.3 企划运营与社区管理
1) **公告板与相册**
- 置顶公告、历史公告留存
- 相册：分类、评论、点赞
- R18：折叠、打码、外链提醒
- 评论可设为**仅企划组可见**

2) **话题管理工具**
- 操作：禁言、删帖、置顶、加精
- 发言频率限制（防刷屏）
- 角色花名：自定义显示名称

3) **互动组件**
- 快速发起：投票、接龙、集体 ASK
- 配置：时间、参与范围
- 结果自动统计与公示

4) **内外场权限分离**
- 内场：核心剧情、任务、互动
- 外场：公开内容、外场专属活动
- 支持内场补员通道

5) **冲突处理机制**
- 操作：禁言、临时小群、转移话题
- 争议投票裁决系统
- 处理日志留存可查

6) **成员参与热力图**
- 后台展示：任务参与、互动频率、强度
- 超负荷玩家预警提示
- 按日/周/企划维度统计

7) **任务/事件依赖配置**
- 前置任务完成率阈值触发
- 特定角色动作触发解锁
- 满足条件自动开放下一内容

8) **动态调参**
- 基于投票/实时数据后台微调
- 参数：伤害量、成功率、概率、冷却
- 实时生效

9) **备用剧情线**
- 参与度低于阈值自动触发
- 轻量剧情，无需高度参与
- 防烂尾：世界观持续

#### 2.3.4 企划组协作与矛盾预防
1) **企划组协作空间**
- 内部任务分配、文件共享、讨论区
- 权限隔离：核心剧情仅指定成员可见
- 操作日志留痕

2) **官方声明发布**
- 类型：澄清、道歉、规则调整、通知
- 历史声明可查、不可篡改
- 支持置顶与推送

3) **玩家举报与反馈**
- 一键举报：违规内容/违规用户
- 自动通知企划组 + 平台后台
- 处理进度可查询

---

### 2.4 互动模块
- **戏群/水群分离**：戏群用 OC 身份发言；水群用主账号发言
- **公共时间线**：展示全员互动帖、作品、公告
- **私信/幕后帖**：私聊、仅参与者可见幕后讨论
- **Bot 系统**：扮演 NPC/GM；触发剧情、发布任务、自动判定、掷骰

---

### 2.5 作品与激励
- **作品展示**：优秀轮播、小红书式信息流
- **作品集归档**：企划结束自动归档，支持一键导出画册
- **虚拟货币**：完成任务获得，可兑换企划内物品
- **称号系统**：按参与度/成就自动发放
- **企友好感度**：互绘、互动、互助累积

---

### 2.6 信用体系
- **信誉分**：企划组可见；低分自动标记、限制参与
- **约稿差评率**：平台监控；过高取消约稿/售卖资格
- **信用记录**：逃单、咕咕、违规、投诉记录

---

### 2.7 趣味化功能
- **吉祥物**：参考多邻国；长时间不交稿状态变化
- **看板娘 + 表情包**：平台形象，用于引导、弹窗、评论区

---

### 2.8 平台后台管理
- **用户管理**：封禁、解封、信息查看、操作日志
- **内容审核**：帖子、作品、人设、公告、评论审核；违规下架
- **数据统计**：用户数、企划数、作品数、互动数、留存
- **数据导出**：支持 Excel/CSV 导出

## 3 盈利功能需求

### 3.1 企划发布费
- 付费企划需审核，通过后收取小额发布费
- 面向高频企划组

### 3.2 高级审核功能
- 解锁：多人协作审核、规则引擎、数据分析
- 面向专业企划组

### 3.3 模板商店
- 付费下载：世界观、人设卡、游戏机制模板
- 创作者与平台分成

### 3.4 周边定制
- 企划结束一键生成画册、实体制品
- 平台抽成

### 3.5 会员订阅
- 企划组版：数据导出、优先客服
- 玩家版：改名、装扮、个性皮肤

---

## 4 技术方案需求（约束）

### 4.1 技术栈（允许二选一）
- **前端**：Flutter / React Native（iOS + Android）
- **后端**：Node.js / Python + MySQL / PostgreSQL
- **第三方**：
  - IM：融云 / 环信
  - 音视频：声网
  - 云存储：阿里云 OSS
  - 推送：极光推送

### 4.2 版本迭代范围（交付边界）
- **MVP（4–5 个月）**：用户系统 + 角色管理；企划核心（主页模板/世界观/人设卡/审核/公告相册）；基础互动 + 作品展示 + 信用体系
- **V1.0（3–4 个月）**：主线/支线任务；机制编辑器 + 沙盒测试；内外场分离 + 协作空间；基础话题管理
- **V2.0（4–5 个月）**：关系图谱、任务依赖、热力图；动态调参、备用剧情线；模板商店、周边定制对接
- **完整版**：全功能上线

---

## 5 非功能需求（强约束）

### 5.1 性能
- 页面打开 ≤ 2s
- 列表加载 ≤ 1s
- 图片上传/预览秒开
- 支持高并发互动与发帖

### 5.2 安全
- 手机号脱敏、用户数据加密
- 内容风控：色情、暴力、侵权、引战
- 防刷票、防作弊、防水军

### 5.3 兼容性
- iOS 13+
- Android 9+
- 主流手机分辨率适配

### 5.4 可用性
- 新手引导流程完整
- 错误提示清晰可操作
- 崩溃率 ≤ 0.1%

---

## 6 验收标准（必须可核对）
1. 所有功能按本文档完整实现，无遗漏
2. 核心流程闭环：注册 → 创建 OC → 创建企划 → 报名 → 审核 → 运营 → 互动 → 归档 → 导出
3. 沙盒测试、内外场、热力图、备用剧情可正常运行
4. 支付、分成、会员、模板商店可正常使用
5. 后台统计、导出、审核、封禁功能可用
6. 满足性能、安全、兼容性指标
